== Description ==
* Author: [Fahad Mahmood](https://www.androidbubbles.com/contact)
* Project URI: <http://androidbubble.com/blog/wordpress/plugins/booking-works>

Booking works is a plugin to manage a platform where users can become vendors and offer their services. Vendors can offer tangible products/services/activities, also online courses/products/services. Vendors can provide bookings for vacations or tours and rental products.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' screen in WordPress

3. There will now be a Split link in the to Woocommerce  order overview page within the order actions on hover.


== Insights ==

* Statistics in one view
* Booking tracking facility
* Add/edit product/services on front-end
* Allow user roles to add/edit product/services
* Total sales on one page
* Add Rental Products / Bookings / On Demand Services
* Add products to Wishlist
* See your recent purchases
* Contact vendor from your order

== Frequently Asked Questions ==

= What is this plugin for? =
Booking works is a plugin to manage a platform where Vendors/Users can rent-out their products and offer their services as professionals. 


= How to become a Vendor? =
Apply to become a Vendor from my-account page and when admin will approve your request, you will become a Vendor user.


= Where I can see all my purchases/orders? =
You can see all your purchases on MY Bills page under my account or order page.


= Can I track my booking/order? =
Yes, you can track your booking/order, but you must be logged-in.
